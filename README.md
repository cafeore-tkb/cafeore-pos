# cafeore-2024 POSシステム

## コマンド

| コマンド                        | 説明                                                              |
| ------------------------------- | ----------------------------------------------------------------- |
| `bun install --frozen-lockfile` | 依存パッケージのインストールには基本的にこれを使ってください      |
| `bun dev`                       | 開発環境の立ち上げ。ホットリロードに対応しています                |
| `bun lint`                      | ESLintの実行。`--fix`オプションをつけることで、自動修正もできます |
| `bun fmt`                       | Prettierの実行                                                    |
